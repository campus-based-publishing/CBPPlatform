getManagementVerbs ( $  verbs = array())

Generic plug-ins implement the enabled/disabled logic by default. This is necessary so that we can make sure that disabled plug-ins will not have to be instantiated.

Call this method with a list of management verbs (if any) generated by the custom plug-in.
