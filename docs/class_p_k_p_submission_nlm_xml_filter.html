Detailed Description

Class that converts a submission to an NLM Journal Publishing Tag Set 3.0 XML document.

FIXME: This class currently only generates partial (citation) NLM XML output. Full NLM journal publishing tag set support will be added as soon as we have migrated document parsing from L8X to the PKP library.
