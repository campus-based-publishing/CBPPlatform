Check (parse and lookup) all raw citations

NB: This handler method is meant to be called by the parallel processing framework (see ProcessDAO::spawnProcesses()). Executing this handler in parallel will significantly improve citation checking performance.

The 'citation_checking_max_processes' config parameter limits the number of parallel processes that can be started in parallel.
