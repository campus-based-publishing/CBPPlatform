& _guessValues ( &$  scoredCitations,
$  scoreThreshold 
)

Take an array of citation parse/lookup results and derive a citation with one "best" set of values.

We determine the best values within the citations that have a score above the given threshold. Citations with a score below the threshold will be ignored.

For these citations we count the frequency of values per meta-data property. The most frequent value will be chosen as "best" value.

If two values have the same frequency then decide based on the score. If this is still ambivalent then return the first of the remaining values.

Parameters:
$scoredCitations
$scoreThresholdinteger a number between 0 (=no threshold) and 100
Returns:
Citation one citation with the "best" values set
