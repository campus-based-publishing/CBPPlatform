Detailed Description

Perform system installation.

See also:
Installer, InstallForm

This script will:

  • Create the database (optionally), and install the database tables and initial data.
  • Update the config file with installation parameters. It can also be used for a "manual install" to retrieve the SQL statements required for installation.
